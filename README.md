# LCP Practitioner Pro Pro Pro

A specialized Python app (Streamlit) designed for Executive Coaches to quickly analyze Leadership Circle Profiles (LCP) and generate a 30-minute debrief roadmap while enforcing strict data privacy (anonymization).

## How to use on Replit

1. Log into your Replit account (replit.com).
2. Create a new Repl and select **Streamlit** as the template.
3. Replace the contents of `main.py` or `app.py` with the code from the `app.py` file in this folder.
4. Add the requirements from `requirements.txt` to the "Packages" section on the left sidebar in Replit.
5. Hit the large green **Run** button at the top!
6. Use the web interface to upload a client LCP PDF, generate your roadmap, and use the Pop Question chat feature.

## Requirements
- Python 3
- Anthropic API Key (Claude)
- Streamlit
- PyPDF2
